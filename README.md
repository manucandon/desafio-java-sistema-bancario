🏦 Desafio: Sistema Bancário em Java
Este projeto é uma aplicação de console desenvolvida como desafio final do curso "Java: Criando a sua primeira aplicação" da Alura. O objetivo foi consolidar conhecimentos fundamentais de lógica, controle de fluxo e interação com o usuário.

🔨 Funcionalidades do Projeto
O sistema simula as operações básicas de uma conta corrente:

Consulta de Saldo: Exibe o valor disponível atualizado.

Depósito (Receber Valor): Permite adicionar créditos à conta.

Transferência (Enviar Valor): Realiza a retirada de dinheiro com validação de saldo (impede que a conta fique negativa).

Menu Interativo: Um loop contínuo que só encerra quando o usuário escolhe a opção "Sair".

Tratamento de Erros: Exibe mensagem de "Opção Inválida" para entradas incorretas.

🚀 Conceitos de Java Praticados
O código utiliza diversos pilares da linguagem:

Scanner: Para leitura de dados via teclado.

Strings Multiline ("""): Para um menu visualmente organizado.

Estruturas Condicionais (if/else): Para as regras de negócio bancário.

Laço While: Para manter a aplicação rodando até o comando de saída.

Operadores de Atribuição Composta (+= e -=): Para atualização eficiente do saldo.


🖥️ Como Visualizar
O código principal está localizado no arquivo Main.java. Para rodar, basta clonar o repositório e executar a classe em qualquer IDE Java (IntelliJ, Eclipse, VS Code).
